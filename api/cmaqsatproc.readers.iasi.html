IASI Ammonia Processor¶

  • bbox subsets each pixel

  • valid = (nh3_total_column > -999) & (AMPM == 0)

  • geometry is a 0.1 degree buffer

  • pixel_area = diameter assumed to be 0.1 degree, which is approximately the 12 km discussed in Clerbaux (doi: 10.5194/acp-9-6041-2009).

  • weights = area
